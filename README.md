# WCPredictor — 2026 FIFA World Cup Ultimate Predictor.AI

Application web de pronostics sur la Coupe du monde 2026 : pronostics entre
joueurs, statistiques scrapées (FBref) et un bot de prédiction (modèle Poisson).

## Stack
React (Vite) · Node.js/Express · PostgreSQL · Python (scraper + IA) · Docker

## Lancer le projet
1. `cp .env.example .env`
2. `docker compose up`
3. Frontend : http://localhost:5173 — API : http://localhost:3000
